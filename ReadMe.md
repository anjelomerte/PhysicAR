# AR-ADL Assessment (HoloLens 2)

This repository contains the source code of a research prototype for an Augmented Reality (AR)–based assessment of activities of daily living (ADL).

The application runs on the Microsoft HoloLens 2 and implements a standardized “vacuuming” task within a real-world environment. It is designed to support research on functional performance in older adults, including individuals with mild cognitive impairment or mild dementia.

The system combines real-world task execution with digital guidance and sensor-based data acquisition. During task performance, the device’s integrated sensors (e.g., inertial measurement, depth sensing, RGB camera, and eye tracking) are used to record movement- and interaction-related information. The aim is to enable differentiated analysis of motor and cognitive aspects of ADL performance within a single, mobile setup.

## Purpose of this Repository

This repository is intended to:

- Document the implementation of the AR-based assessment prototype  
- Support transparency and reproducibility of the technical approach  
- Provide a reference implementation for related research projects  

The associated research manuscript describing the conceptual background and evaluation framework is currently under preparation.
